NEBULA PLAY - VERSION FINAL HASTA EL MOMENTO

1) Abrir PostgreSQL/pgAdmin y verificar que exista la base BDNebulaPlay.

2) Si ya tienes la base original importada:
   - Abre pgAdmin > BDNebulaPlay > Query Tool
   - Ejecuta: database/seed_empresas_vacantes_candidatos.sql

3) Si quieres crear todo desde cero:
   - Crea una base nueva llamada BDNebulaPlay
   - Ejecuta: database/BDNebulaPlay_con_datos_demo.sql

4) Configurar el archivo .env en la raiz del proyecto:
   DATABASE_URL=postgresql://postgres:hola@localhost:5432/BDNebulaPlay
   PORT=4000
   SESSION_SECRET=nebula_secret_123

5) En VS Code, abre la carpeta nebula-play y ejecuta:
   npm install
   npm run dev

6) Abrir en navegador:
   http://localhost:4000
